# Student To Do App

A simple and interactive web application designed to help students organize their academic tasks efficiently. Built with **HTML**, **CSS**, and **JavaScript**, this app allows users to add, manage, and track their daily to‑dos in a clean interface.

---

## ✨ Features
- ➕ Add new tasks quickly
- ✅ Mark tasks as completed
- 🗑️ Delete tasks when no longer needed
- 🎨 Clean and responsive UI
